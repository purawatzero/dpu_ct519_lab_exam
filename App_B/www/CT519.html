สรุปความรู้ที่ได้จากวิชา CT519

รู้จักการใช้งานบน Cloud และได้ทดลองใช้งานเองครั้งแรก โดยต้องมีการฝากข้อมูลไว้กับผู้ให้บริการที่สะดวกสบายมากขึ้นในการใช้งาน และความรู้ที่ได้มาจะมาปรับใช้งานในองค์กรเพื่อให้เกิดประโยชน์ให้มากที่สุดเท่าที่จะทำได้ รวมทั้งการใช้งาน Virtual Networking ซึ่งความรู้ส่วนนี้ไม่ได้ใช้งานมานานแล้ว เพราะเมื่อก่อนเคยแค่ config network เท่านั้น ซึ่งได้มาต่อยอดในส่วนนี้ที่ทำให้ความรู้แน่นขึ้น และความรู้เรื่อง Docker ที่ทางบริษัทเคยเปิดสอนในที่ทำงาน แต่ได้ความรู้จากที่ได้เรียนเพิ่มเติมจากเดิมจาก CT519 มากขึ้น ทำให้ความรู้ที่ได้มาจะนำไปต่อยอดในแผนกตัวเองให้เกิดประโยชน์มากขึ้น
